## Способы решения задач ЦАП  
ЦАП не только проверяет код на наборах тестов, но и анализирует тексты программ
алгоритмом классификации. Для задач разных типов алгоритмом кластеризации были
выявлены наиболее распространённые в предыдущем году обучения подходы к решению
задач, на полученных данных был обучен классификатор на основе искусственной
нейронной сети, определяющий способы решения. Ниже приводится перечень способов
решения задач ЦАП.
